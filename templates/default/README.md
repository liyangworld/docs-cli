### 开始开发

```bash
# 安装依赖
$ yarn # 或者 npm install

# 本地开发
$ npm run dev

# 打包
$ npm run build

# 发布
# 安装 deploy-this
$ yarn global add deploy-this # 或者 npm install deploy-this -g
$ deploy publish dev -m 'v1.1.0 发布信息'
```
